# Exploratory Data Analysis – Smartphones Dataset

## Objective
To analyze smartphone pricing trends and understand feature relationships.

## Dataset Description
The dataset contains 15,000 smartphone models with attributes such as price, RAM, battery capacity, storage, and brand.

## Descriptive Statistics
- Total Models: 15000
- Average Price: ₹43,470
- Minimum Price: ₹5,000
- Maximum Price: ₹1,49,976

## Key Insights
- Most smartphones fall under mid-range pricing.
- RAM has positive correlation with price.
- Battery capacity has weak impact on pricing.
- Premium segment phones are above ₹80,000.

## Conclusion
Smartphone pricing is influenced mainly by RAM and premium positioning.
